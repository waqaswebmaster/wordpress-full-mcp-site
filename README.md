# WordPress Full MCP Website

Marketing and documentation site for the **WordPress Full MCP** plugin.

## Pages

- `/` — Home
- `/about` — About
- `/contact` — Contact
- `/docs/*` — Full documentation

## Develop

```bash
cd website
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Static output lands in `website/dist/`. Deploy that folder to any static host (Netlify, Vercel, Cloudflare Pages, GitHub Pages, etc.).
